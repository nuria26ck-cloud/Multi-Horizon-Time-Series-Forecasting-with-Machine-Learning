# Data Folder

Place the Headge fund dataset here after downloading from Kaggle: https://www.kaggle.com/competitions/ts-forecasting

This folder is intentionally left empty due to file size constraints.

## Loading data: 

Before running any code, make sure you load the dataset properly by pointing the pathway
to your dataset file correctly, and ensuring the name of the .parquet files are consistent throughout your code. 

The data is already split, so when .zip file is downloaded you should have two .parquet files: test.parquet and train.parquet for your training dataset and holdout/ testing data set.
